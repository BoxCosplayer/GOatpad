# Goatpad
An extensible text editor built for fast, keyboard-driven workflows with a goat-themed vibe.

## What This Is
- **Purpose:** A personal-but-extensible editor inspired by vim/neovim, optimized for Windows.
- **Audience:** Primarily for the author, but open to community feedback, issues, and pull requests.
- **Philosophy:** Local-first, CLI-first, and themed with Goatpad flair.

## Platform & Constraints
- Operates offline/local by default.
- Designed to favor CLI over GUI for configuration and interaction.

## Core Features (Required)
- Folder-aware editing with integrated search across files and documents.
- Embedded terminal support.
- Command macros reminiscent of vim/neovim.
- CLI-first feature set with configurable settings, themes, and preferences.
- Goat-themed customization options.

## Nice-to-Have Features (Planned)
- Git and diff integration.
- Language linting support.
- Discord presence/integration.
